Welcome to my GitHub repository for the JHU PMB R and ggplot2 code demo!

The code and files in this repository can be used to follow along with my presentation. Feel free to download the items here and play with them at your leisure. There is a nice, html format of the demonstration included. You can open it in RStudio and then click "Run Document" (RStudio may have to download some packages).
